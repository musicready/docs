## Duration Codes

| Code | Duration | Beats (4/4) |
|------|----------|-------------|
| `w` | Whole | 4 |
| `h` | Half | 2 |
| `q` | Quarter | 1 |
| `e` | Eighth | 1/2 |
| `s` | Sixteenth | 1/4 |
| `t` | 32nd | 1/8 |
| `x` | 64th | 1/16 |

Add `.` for dotted, `..` for double-dotted.

## Required Properties (in MRS-S storage)

### Event Properties

| Property | Description | Computed By |
|----------|-------------|-------------|
| `:id` | Stable UUIDv7 identifier | Orchestrator |
| `:at` | Absolute beat position | Orchestrator (derived) |

### Measure Properties

| Property | Description | Computed By |
|----------|-------------|-------------|
| `:id` | Stable UUIDv7 identifier | Orchestrator |
| `:number` | Display number (human navigation) | Orchestrator |
| `:beat-start` | Absolute beat position | Orchestrator (derived) |

## Agent Output (MRS-Ops)

Agents provide position relative to measure; orchestrator computes absolute position:

| Agent Provides | Orchestrator Derives |
|----------------|----------------------|
| `:beat` (relative to measure) | `:at` (absolute position) |
| `:tmp-id` (temporary identifier) | `:id` (UUIDv7) |

## Optional Event Properties

| Property | Values | Description |
|----------|--------|-------------|
| `:dyn` | pp, p, mp, mf, f, ff, etc. | Dynamic marking |
| `:art` | staccato, tenuto, accent, etc. | Articulation |
| `:orn` | trill, mordent, turn, etc. | Ornament |
| `:tech` | pizz, arco, legno, etc. | Playing technique |
| `:stem` | up, down | Stem direction |
| `:grace` | true/false | Grace note |

## Lane Bundles

| Bundle | Lanes | Use Case |
|--------|-------|----------|
| `orchestrate` | notes, expression, technique | Write parts from sketch |
| `dynamics-pass` | expression | Add dynamics to existing music |
| `notation-cleanup` | notes, technique | Fix notation details |
| `full-compose` | structure, temporal, harmonyPlan, notes, expression, technique | Complete composition freedom |

## Context View Types

| View Type | Content | Use Case |
|-----------|---------|----------|
| `melodic-reference` | Melody to respond to | Counterpoint, accompaniment |
| `harmonic-context` | Chord progression | Voice leading |
| `orchestration-map` | Active instruments, texture | Scoring decisions |
| `phrase-structure` | Phrase boundaries | Phrasing decisions |
| `dynamics-profile` | Dynamics trajectory | Dynamic shaping |
| `thematic-references` | Themes and motifs | Thematic development |

## Working Set Scope Fields

| Field | Required | Description |
|-------|----------|-------------|
| `:measures` | Yes | UUID range (inclusive) |
| `:instruments` | Yes | List of instrument IDs to include |
| `:voices` | No | List of voice IDs (default: all) |
| `:display-hint` | No | Human-readable measure numbers |
| `:source-hash` | No | SHA-256 for conflict detection |
| `:task-context` | No | Human-readable task description |
| `:constraints` | No | Validation rules for agent output |
